Pokemon Radical Red Walkthrough

Chapter 0 - Setup

- Download Emulator: https://visualboyadvance.org/download/1-8-0-beta-3

- Download FireRedRom:https://archive.org/details/1636PokemonFireRedUSquirrels

- Create Desktop Folder called "Radical Red" and place both downloads in it.

- Use this patcher on FireRed to turn it into Radical Red 4.0: https://patch.radicalred.net/

- Rename the patched FireRed file to "Radical Red 4.0",
to ensure multiple types of gameplays, make copies of the Radical Red 4.0.

- Name one file as "rr4.0 with friends", and the others you can just leave as "Radical Red 4.0".
